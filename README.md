# US National Debt Analysis (Excel)

## Overview
This project analyzes U.S. national debt data to identify long-term trends and changes over time using Excel.

## Tools Used
- Microsoft Excel

## Key Analysis
- Tracked national debt growth over multiple years
- Identified upward and downward trends
- Organized and cleaned raw economic data for analysis
- Created structured views to support trend interpretation

## Outcome
The analysis highlights long-term debt growth patterns and provides a clear view of how national debt has evolved over time.
